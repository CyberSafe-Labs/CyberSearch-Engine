How To Make Us Your Default Search Engine

Firefox Based Browsers (Firefox, Waterfox, Cliqz, etc)

Desktop

  1. Go to https://cybersearch.herokuapp.com, click on the green plus on the search bar (shown in the image below) and click on Add "CyberSearch".
  2. Doing this only adds CyberSearch to the list of search engines available in Firefox. To make us your default search engine, go to about:preferences#search and change the default search engine to CyberSearch (as shown in the image below).

Mobile

On mobile, all that you need to do is go to your settings, press the search option, then press add search engine, and set the URL area to https://cybersearch.herokuapp.com/results?q=%s.

Chrome Based Browsers (Chrome, Brave, Opera, etc)

  1. Go to settings
  2. Go to search engines
  3. Press manage search engines
  4. Press add
  5. Add the information shown below
The search engine and keyword value can be set to anything but the URL has to be https://cybersearch.herokuapp.com/results?q=%s

Vivaldi

  1. Go to settings
  2. Go to search
  3. Press the + button
  4. Add the information shown below
The search engine and nickname value can be set to anything but the URL has to be https://cybersearch.herokuapp.com/results?q=%s

Safari

Currently on Safari, there is no option to add custom search engines. Hopefully they change this one day.

Other Browsers

If your browser is not listed above and you need more instructions, please contact us and we will update our instructions to include your browser.
